# Stored Procedures Exercise (SQL)

## Scenarios Covered

1. **Process Monthly Interest**
   - Applies 1% interest to all savings accounts.
   - File: `1_ProcessMonthlyInterest.sql`

2. **Update Employee Bonus**
   - Adds a bonus % to employees in a specified department.
   - File: `2_UpdateEmployeeBonus.sql`

3. **Transfer Funds**
   - Transfers a specified amount between two accounts.
   - File: `3_TransferFunds.sql`

## Instructions

1. Run each `.sql` file separately in MySQL Workbench, phpMyAdmin, or any compatible SQL environment.
2. Each file creates required tables, inserts sample data, creates the procedure, calls it, and shows the updated output.
3. Enable DBMS output display if applicable (for tools like Oracle SQL Developer).
